Question-1: What are the different ways to select an element in the DOM?
There are different ways to select an element in the DOM such as: getElimentById(), getElementsByclass(), getElementByTagName(),querySelector() etc.
Question-2: What is the difference between innerHTML, innerText, and textContent ?
innerHTML is used to get html inside element. innerText is used to set new text. textContent is used when we need to catch all text
Question-3: What is event delegation in the DOM?
Event delegation is a technique in JavaScript where instead of adding event listeners to individual child elements, we add a single event listener to a parent element. The event then bubbles up from the child elements to the parent
Question-4: What is event bubbling in the DOM?
Event Bubbling in the DOM is a process where event starts from the target element and bubble up to the root element
Question-5: How do you create, add, and remove elements using JavaScript?
We can creat a new element in JavaScript by using the document.createElement(). We can add element by using appendChild() and can remove an element by using removeChild()
